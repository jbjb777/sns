메일에 반드시 포함해야 하는 내용

  1. 제목 : [개인정보 삭제 요청] – 본인 닉네임 또는 아이디
  2. 본문에 포함할 정보 :

    • 사용 중인 아이디 (로그인할 때 사용하는 ID)

    • 삭제를 원하는 범위 (계정 전체 삭제 / 특정 게시물만 삭제 등)
    • 추가 확인용 정보 (예: 가입한 대략적인 날짜, 최근 접속 날짜 등)
  3. 가능하면 사용하던 기기나 브라우저 정보도 함께 적어주시면 더 빠른 확인이 가능합니다.

처리 소요 기간 : 요청 메일 접수 후 보통 3~7일 이내 처리 완료되며,
완료 시 요청하신 이메일로 결과를 알려드립니다.

참고 : 일부 법령에 따라 보관 의무가 있는 최소한의 로그 정보(예: 접속 기록 등)는 삭제 대상에서 제외될 수 있습니다.
